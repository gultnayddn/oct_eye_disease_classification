# OCT Eye Disease Classification

This project focuses on classifying OCT retinal images into four categories:
- CNV
- DME
- DRUSEN
- NORMAL

## Dataset
OCT2017 dataset from Kaggle.

## Model
- CNN with MobileNetV2
- Transfer Learning
- TensorFlow / Keras

## Features
- Training & evaluation
- Confusion matrix & metrics
- Random image inference
- Gradio-based user interface

## How to Run
```bash
pip install -r requirements.txt
python app_gradio.py
